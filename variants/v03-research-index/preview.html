Research avenues

04 primary · 01 supporting
01/ Explainability

Explainability & Interpretability

Methods that reveal how code models reason — rationales, traces, tests — so developers can inspect, trust, and debug automation.

→
02/ Efficiency

Efficiency & Resource-Aware Automation

Quantization, distillation, and PEFT to deliver low-latency, memory-light, cost-aware automation measured on real workloads with reproducible metrics.

→
03/ Multi-Agent

Multi-Agent LLMs for SE

Role-aligned agents that plan, code, test, and review — coordinated to produce coherent, end-to-end developer assistance.

→
04/ Neurosymbolic

Neurosymbolic AI for SE

Hybrid pipelines pairing LLMs with program analysis and constraints to produce interpretable, checkable fixes, tests, and designs.

→
05/ Supporting

Task-Aware Code Automation

From issues, diffs, and traces to concise summaries, TODOs, and plans — models that reduce developer toil with actionable, repo-aware guidance.
